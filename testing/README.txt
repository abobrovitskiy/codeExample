Программа сортировки данных в файле.
Сборщик проекта maven
Версия java 1.8.161
Для запуска программы нужно запустить bat файлы с соответствующим названием.
JAR файл находится в папке testing\chapter_001\target\filesorter.jar
Входной файл с числами testing\chapter_001\target\in_int.txt
Входной файл со строками  testing\chapter_001\target\in_str.txt
Выходной файл out.txt формируется каждый раз заново.
